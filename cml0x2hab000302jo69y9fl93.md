**What is HTML ?**

**HTML stands for HyperText Markup Language. It is like a skeleton of webpage. We create webpage, web application with the help of HTML. It has multiple tags through which we add heading, paragraph, image, audio, video etc in our webpage.**

**Tag in HTML :**

**Tag is like a label, each label has a particular task e.g. &lt;p&gt; tag is use to add paragraph, &lt;img&gt; tag is use to add image, &lt;b&gt; tag is use to bold the text. We have a number of tag for different tasks.**  

**What an HTML Element Means:**

**HTML Elements is the combination of opening tag, closing tag and the content which we write inside the tag e.g.**  &lt;h1&gt;Hello World&lt;/h1&gt;. We can say this is element.

1. Self-closing (void) elements
    

There are some elements which doesn’t needs to close. They are self closing elements.

Example: &lt;br&gt; --to break the line, &lt;hr&gt; to create a horizontal line, &lt;img&gt; etc.

**Block-Level vs Inline Elements**

**Block-level elements**: Start on a new line, take up full width.  
Examples are &lt;div&gt;, &lt;p&gt;, &lt;h1&gt;

**Inline elements**: Stay within a line, only take up as much space as needed.  
Examples: &lt;span&gt;, &lt;a&gt;, &lt;strong&gt;

**Commonly Used HTML Tags**

* &lt;h1&gt; to &lt;h6&gt; → Headings
    
* &lt;p&gt; → Paragraphs
    
* &lt;div&gt; → Division/section
    
* &lt;span&gt; → Inline text container
    
* &lt;a&gt; → Links
    
* &lt;img&gt; → Images
    
* &lt;ul&gt;, &lt;ol&gt;, &lt;li&gt; → Lists
